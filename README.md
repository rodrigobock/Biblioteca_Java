# Biblioteca_Java

Sistema desenvolvido em Java voltado para o gerenciamento de uma biblioteca, focando em conceitos de Orientação a Objetos.

## Tecnologias
- Linguagem: Java
- Ambiente: Projetos configurados para Eclipse IDE.

## Como Executar
1.  Certifique-se de ter o JDK (Java Development Kit) instalado em sua máquina.
2.  Importe o projeto para o Eclipse (File > Import > General > Existing Projects into Workspace).
3.  Navegue até a pasta `src` e localize as classes principais (ex: `core`, `livro`, `testes`) para execução.
